Мастер пол - это веб-приложение, разработанное для автоматизации взаимодействия компании с партнерами, 
осуществляющими доставку продукции до конечных потребителей. Система обеспечивает полный цикл работы с партнерами: 
от ведения базы данных до анализа эффективности сотрудничества.

***Функциональные возможности***

===Управление партнерами===
- Просмотр списка партнеров с основной информацией
- Добавление новых партнеров через интуитивную форму
- Редактирование данных существующих партнеров
- Загрузка логотипов компаний-партнеров

===Аналитика продаж===
- История реализации продукции по каждому партнеру
- Фильтрация данных по датам и периодам
- Визуализация статистики продаж

===Расчетные функции===
- Автоматический расчет скидок на основе объема продаж

Гибкая система скидок:
- До 10,000 шт. — 0%
- От 10,000 до 50,000 шт. — 5%
- От 50,000 до 300,000 шт. — 10%
- Свыше 300,000 шт. — 15%

===Управление производством===
- Расчет материалов для производства продукции
- Учет брака материалов в производственном процессе
- Оптимизация расходов на материалы

***Технический стек***

===Backend===
- Python 3.14
- Django 5.2.8
- PostgreSQL 

===Frontend===
- HTML5 с шаблонами Django

===База данных===
- Реляционная структура в третьей нормальной форме (3НФ)
- Оптимизированные запросы
- Согласованная схема именования
